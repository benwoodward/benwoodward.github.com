Overview

Working with APIs I do a lot of manual testing on the command line using cURL. Typing out -H "Accept: application/json" -H "Content-Type: application/json" time after time can get tiresome. Here I will explain a simple bash function I use that will pretty print, and colourize the JSON output in your Terminal. This tutorial is written for intermediate developers using Macs, however, I’m sure all these commands will work fine on Linux machines. Let me know in the comments if you have a problem.
